# **App Name**: EcoVend Hub

## Core Features:

- User Authentication: Secure user signup and separate login for users and admin roles.
- Reverse Vending Machine Interface: Interface to record transactions when plastic bottles are deposited, track points earned, and manage user balances.
- Rewards Catalog: Display available rewards that can be redeemed with points.
- Point Redemption: Allow users to redeem points for selected rewards.
- Admin Dashboard: Dashboard for administrators to manage machines, users, and reward inventory.

## Style Guidelines:

- Primary color: Emerald green (#4CAF50) to represent ecology and sustainability.
- Background color: Light pale green (#F1F8E9), same hue family but much lighter.
- Accent color: Golden yellow (#FFC107) for highlighting rewards and calls to action; analogous to green but significantly brighter and less saturated.
- Font: 'PT Sans', a sans-serif typeface suitable for both headlines and body, lending the app a clean, friendly and readable aesthetic.
- Use recycling-themed icons for bottle deposit, point balance, and rewards.
- Clean and intuitive layout with clear calls to action for depositing bottles and redeeming points.
- Subtle animations for reward redemption confirmation.
